# Aurelia Bakehouse

A fully responsive, editorial artisan bakery storefront built with semantic HTML, modern CSS and vanilla JavaScript.

## Preview

Open `index.html` directly, or serve the folder locally:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.

## Included interactions

- Transparent-to-blurred sticky navigation and responsive mobile menu
- Product category filtering and expandable full menu
- Searchable product catalogue
- Persistent basket with quantity editing and checkout request flow
- Custom cake consultation form
- Delivery PIN code checker
- Testimonial carousel and gallery lightbox
- Newsletter validation, scroll reveals and reduced-motion support

All imagery and fonts used by the page are bundled locally for a fast, network-independent preview.
